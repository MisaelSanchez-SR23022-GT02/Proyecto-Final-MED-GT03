# Documentación de Proyectos Académicos: Sistemas de Archivos y Planificación

Este repositorio contiene la implementación y análisis de dos proyectos fundamentales de sistemas operativos y estructuras de datos: un Mini Sistema de Archivos basado en Árboles B+ y un Planificador de Procesos con Colas Multinivel.

## Descripción de los Proyectos

### Proyecto 1: Mini Sistema de Archivos (Árbol B/B+)
Enfoque en la representación de directorios y archivos mediante estructuras de datos jerárquicas balanceadas.
* **Objetivos:** Simular operaciones de manipulación de nodos, visualización de reorganización del árbol y justificación de su uso en sistemas reales.
* **Entregables:** Diagramas operativos, pseudocódigo detallado y análisis de complejidad Big-O.

### Proyecto 2: Planificador de Procesos (Colas Multinivel)
Implementación de algoritmos de planificación para optimizar el rendimiento del sistema.
* **Objetivos:** Simulación de algoritmos Round Robin, gestión por Prioridades y colas multinivel retroalimentadas.
* **Entregables:** Diagramas de colas, cálculo de tiempos de espera/retorno y análisis de eficiencia Big-O.
